# [license_plate_recognition image processing project about license plate recognition - araç_plakası_tanımlama görüntü işleme projesi]

## 📌 Overview - Genel Bakış
- This repository features an Python based license_plate_recognition project which can extract plate numbers from full car images. 
- Bu kod tabanı, araç görüntülerinden plaka numaralarını çıkarabilen Python tabanlı araç plakası tanımlama projesidir.  

## 🚀 Key Features - Anahtar Özellikler
- **Image Processing - Görüntü İşleme:** Place image is created with gray coloring, cropping and compressing.
  Gri tonlamalı görüntüleme, kırpma ve sıkıştırma yöntemleriyle plaka görüntüsünü alınır.  
- **Plate Recognition - Plaka Tanımlama:** Plate numbers are extracted and best plate results are ordered. Plaka numaraları çıkarılmaktadır ve en iyi plaka sonuçları sıralanmaktadır.
